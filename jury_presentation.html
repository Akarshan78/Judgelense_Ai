1. JudgeLens AI: Master Specification

JudgeLens AI is an advanced, multi-agent predictive analytics platform designed to simulate elite hackathon judging. This document serves as the absolute source of truth for the project's architecture, logic, and operational philosophy.

The AI Persona Engine

JudgeLens utilizes Persona-Driven Engineering to provide a "Judgement Panel" effect.

  • The Judge: The Academic. Provides objectivity and rubric alignment.
  • The Skeptic: The VC Shark. Mimics the "No" phase of venture funding. Provides brutal honesty.
  • The Mentor: The Architect. Provides a constructive path forward and actionable steps.
  • The Historian: The Librarian. Provides context and comparisons to winning projects.

Technical Stack

  • Frontend: React 19, Vite, Tailwind CSS, Recharts, Framer Motion (CSS animations).
  • Backend: FastAPI (Python), Google Gemini 2.0 Flash, Pydantic, Asyncio.
  • Resilience: Exponential backoff for rate limits, sequential orchestration for burst prevention.

4. Developer's Manual

This section explores the "Code-Level" reality of JudgeLens AI.

The AI Engine: Prompt Engineering

The backend logic resides in `backend/agents.py`. The `safe_generate_content` function implements a Resilience Loop to handle API instability.

async def safe_generate_content(prompt: str, retries=3, delay=10):
    for attempt in range(retries):
        try:
            response = await client.aio.models.generate_content(...)
            return response
        except Exception as e:
            if "429" in str(e): # Rate Limit Hit
                await asyncio.sleep(delay * (attempt + 1)) # Exponential Backoff

5. Comprehensive File Analysis

A detailed breakdown of every critical file in the repository.

Backend Architecture (`/backend`)

main.py

The entry point for the FastAPI server.

  • Initializes FastAPI: Sets up the app with title and metadata.
  • CORS Middleware: Configures `CORSMiddleware` to allow requests from the React frontend.
  • Dependency Injection: Initializes the `Orchestrator` singleton.
  • Endpoints:
    • `POST /analyze`: Accepts `ProjectSubmission`, validates header keys, calls the Orchestrator, and returns `EvaluationResult`.
    • `GET /`: Health check endpoint.

agents.py

The core intelligence layer. Contains all LLM interactions.

  • Config: Loads environment variables (`GEMINI_API_KEY`) and configures the `google-genai` client.
  • Helper: `safe_generate_content()` implements exponential backoff for 429 errors.
  • Agents:
    • `JudgeAgent`: Structured JSON scoring.
    • `SkepticAgent`: Adversarial critique.
    • `MentorAgent`: Improvement suggestions.
    • `BenchmarkAgent`: Historical comparison.
  • Orchestrator: Manages the sequential execution flow (`await asyncio.sleep(5)`) to prevent rate limiting.

models.py

Pydantic data definitions for type safety.

  • `ProjectSubmission`: `title`, `description`, `techStack`.
  • `EvaluationResult`: The complex nested structure returned to the UI, including scores, prediction, and critique.

Frontend Architecture (`/`)

App.tsx

The main application controller and layout.

  • State Management: Uses `useState` to track `AppStatus` (IDLE, EVALUATING, RESULTS, ERROR).
  • Layout: Renders the Navbar, Hero Section, submission form, and results conditionally.
  • Styling: Implements the glassmorphism container and dynamic background elements.

components/SubmissionForm.tsx

The user input interface.

  • Validation: Ensures title and description are not empty.
  • Visuals: Uses glassmorphism inputs and a gradient submit button with loading spinner state.

components/EvaluationResults.tsx

The data visualization dashboard.

  • Recharts: Renders the Radar Chart for category dimensions.
  • Verdict Ring: SVG-based animated ring showing the overall score.
  • Critique Card: Displays the persona-driven feedback.

services/geminiService.ts

The API communication layer.

  • Fetcher: Sends `POST` requests to the backend.
  • Error Handling: parses non-200 responses and throws descriptive errors for the UI to catch.

Configuration & Styles

index.html

The entry HTML file.

  • Global Styles: Contains the CSS for `.glass`, `.accent-gradient`, and the `.grid-pattern` animation.
  • Fonts: Imports 'Plus Jakarta Sans' from Google Fonts.
  • Tailwind: Loads Tailwind via CDN for rapid styling (in this specific setup).

.env & .env.local

Environment variable management.

  • `.env`: Stores the `GEMINI_API_KEY` for the backend.
  • `.env.local`: Stores frontend-specific variables (if any).
